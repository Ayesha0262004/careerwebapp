## 📚 Skills Arena - Learn Topics

### How to Navigate
```
Skills Arena (/skills)
├─ Subject Tabs (Python, SQL, Aptitude)
├─ Topic Progress Display
├─ Topic List with Status
│  ├─ ✓ Completed (green)
│  ├─ → In Progress (yellow)
│  └─ ✗ Locked (gray)
├─ Difficulty Badges
├─ XP Point Display
└─ Achievement Section
```

### Step-by-Step Learning

**Step 1: Choose Subject**
- Click Python, SQL, or Aptitude tab
- View overall subject progress

**Step 2: Pick Topic**
- Read topic name and description
- Check difficulty level
- See how many XP you'll earn

**Step 3: Learn**
- Click "Start" button
- Read materials & watch videos
- Take notes if needed

**Step 4: Test Knowledge**
- Complete quiz (aim for 70%+)
- Review wrong answers
- Retry if needed

**Step 5: Track Progress**
- Topic marked as complete
- XP added to total
- Progress bar updates

### Pro Tips
- Start with Easy topics
- Build momentum
- Review before moving next
- Unlock interview questions at 70%
- Try to master all topics in phase

## 📐 Formula Vault - Quick Reference

### Finding Formulas

**Option 1: Browse by Category**
1. Click category tab (Geometry, Algebra, etc.)
2. View all formulas in that category
3. Click any formula for details

**Option 2: Search**
1. Type in search box
2. See matching results
3. Filter by difficulty

### Understanding Formulas

```
Formula Card
├─ Formula Name
├─ Mathematical Expression
├─ Plain Language Description
├─ Real World Example
├─ Difficulty Level
└─ Action Buttons
   ├─ Copy to Clipboard
   ├─ Download PDF
   └─ Bookmark/Save
```

### Using Bookmarks

**To Save Formula:**
1. Click star icon (⭐)
2. Formula added to bookmarks
3. View in "Saved Formulas" section

**To Copy:**
1. Click "Copy" button
2. Paste in notes/documents
3. Ready to use

**To Download:**
1. Click "Download"
2. Save as PDF
3. Print or view offline

### Formula Tips
- Bookmark frequently used ones
- Review before exams
- Copy to notes
- Create personal formula sheet
- Practice with examples
